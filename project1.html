Street Cameras Installation

Sep. 2019 - Dec. 2019

- Designed a system to help police department with the installation of security cameras at traffic intersections, which supports adding/changing/removing streets operation and generating street graph to compute the optimal installation location of cameras.

- Used advanced Linux programming for multiprocesses to drive all parts working in order in C++.

- Improved human-machine interface by increasing system error tolerance and the efficiency of system effectiveness and correctness tests by building an extra random qualified streets generation system.

- Optimized the performance of the system on large input by developing the algorithm to trade off time complexity and approximation ratio, such as reducing running time sharply while reaching 90% accuracy.
